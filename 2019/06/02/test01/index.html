—002 NAT 网络地址转换
安全 私有地址 转换 公有地址(共享上网) 共用一个IP

安全 不愿意 让内部的电脑 直接暴漏在公网上

端口号 一个端口号 = 一项服务

1台路由器 如何 伪装成pc？
1.关闭路由功能 【no ip routing】
2.接口配置ip,掩码，
3.配置网关(网关配置：ip default-gateway 192.168.1.1)

PBR — Policy Based Routing 策略路由
三步 NAT操作步骤：
1.进入外网口 连接ISP接口 开启NAT转换出去的接口
interface f1/0
ip nat outside
2.进入内网口 连接server接口 开启
interface f0/0
ip nat inside
3.配置公网地址和内网地址的关系
ip nat inside source static tcp 192.168.1.2 23 100.1.1.2 4001
telnet 100.1.1.2 4001

telnet 属于tcp服务，基于可靠的传输方式

type rotary — 轮询 循环

TCP – PAT 负载均衡 —> 只能针对TCP的数据 进行负载均衡
ip nat outside
ip nat inside

access-list 1 premit any  --通过ACL 匹配任何来源
ip nat pool NAT 192.168.1.10 192.168.1.12 netmask 255.255.255.0 type rotary
ip nat inside destination list 1 pool NAT
